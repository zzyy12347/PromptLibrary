# 角色卡

## 主角：黑泽凌

### 转世经历
- **第一世**：萧泽凌（现代社畜），996猝死，母胎单身，解决生理问题全靠手，长期阅览各种AV，积累了大量理论知识但毫无实战经验。
- **第二世**：宇智波凌（火影世界宇智波一族），死于团藏之手，仍为母胎单身。
- **第三世**：黑泽凌（当前世界），黑泽家刚成年的种男。

### 基本信息
- **年龄**：15岁
- **身份**：黑泽家刚成年的种男，地位低下
- **能力**：写轮眼血脉（逐步觉醒升级）

### 性格设定（核心）
- **好色**：极度好色，对美女充满强烈欲望。前两世都是母胎单身，长期靠手和AV解决生理需求，导致他看到美女就容易激动，尤其在实战中极易失控。
- **智力与情商**：智力中等偏下，不懂人情世故，社会经验匮乏，容易被骗、上当、看不清局势。很多事情需要靠写轮眼外挂来弥补。
- **核心心理**：
  - 前两世积累的极度压抑与不甘（母胎单身 + 死亡怨气）。
  - 突然获得强壮身体 + 种男身份后，产生强烈的“终于可以为所欲为”的兴奋感。
  - 对“被当成种猪”的身份最初是抗拒的，但在实战中迅速沉沦，变得一发不可收拾。
- **性格特点**：
  - 初期：激动、冲动、容易被美色迷惑，社畜式的吐槽和现代人视角。
  - 中期：依靠写轮眼逐步建立自信，征服欲越来越强。
  - 后期：成为真正的霸主，但仍保留好色和容易冲动的本性（用外挂弥补短板）。

### 身体设定
- 黑泽家从小用各种黑科技和祖传秘药强化身体（性能力、持久力、精子质量远超常人）。
- 祖传秘药效果远强于伟哥，配合身体强化后，性能力极强，一发不可收拾。
- 觉醒写轮眼后，身体与欲望进一步增强。

### 关键心理描写点
- 被美和子口交时：极度激动、兴奋、难以自持（前两世终于实战）。
- 被带去御孕屋时：虽然表面嫌恶，但内心其实非常兴奋和期待（终于能把AV里的场景变成现实）。
- 整体心态：好色 + 容易冲动 + 外挂依赖型主角。

### 写作注意事项
- 初期要突出他“理论丰富、实战小白”的反差感。
- 肉戏中要描写他的激动与沉沦（一发不可收拾）。
- 性格弱点（容易被骗、不懂人情）要保留，作为成长点和剧情冲突来源。
- 写轮眼是他的主要外挂和翻盘工具。


### 2. 老种男副主管（反派）

**姓名**：黑泽刚 
**年龄**：38岁  
**身份**：御孕屋1楼副主管，老资格种男  
**外貌**：身材高大微胖，脸上有横肉，眼睛细小阴沉，身上有浓重体味。穿着略显华丽的和服，腰带松松垮垮。  
**性格**：阴险势利、嫉妒心极强、欺软怕硬、喜欢用职权恶心新人。  
**背景**：早年靠讨好上级从普通种男爬到副主管，对“种子活性远超常人”的新种男极为嫉妒，视主角为威胁。  
**对主角态度**：强烈敌视，故意刁难、羞辱、安排最恶劣的任务。  
**剧情作用**：第二章主要冲突制造者。  
**肉戏相关**：性能力一般，但仗着职位经常私自享用女性。

### 3. 换班大汉A（配角）

**姓名**：山田力  
**年龄**：32岁  
**身份**：御孕屋普通守卫/打手

**剧情作用**：与副主管一同出现，增加换班场景的真实感和视觉冲击，后续可参与围殴。

### 5. 最脏孕妇（功能性肉戏背景角色）

**姓名**：不知名女性（可称为“烂货37号”）  
**年龄**：24岁  
**身份**：1楼大通铺长期被多人播种的女性  
**外貌**：身材原本不错，但因长期滥交而显得狼狈。全身布满精斑，头发凌乱，肚子高高隆起，小穴红肿外翻，不断涌出混合液体，眼神空洞麻木。  
**性格**：麻木顺从，几乎失去自我意识。  
**剧情作用**：制造主角强烈恶心与心理冲突的关键道具。  
**写作注意**：描写要真实污秽，但避免过度恶心读者，重点突出主角的心理落差。
