# 《轮回种王：写轮眼征服录》完整大纲

**作者备注**：模块化设计，方便连载写作与随时插入新点子。核心设定为**三次转世** + 从头高强度后宫 + 写轮眼血统征服流。

## 主角转世设定
- **第一世**：萧泽凌（现代社畜），996猝死，母胎单身，全靠手和AV解决生理需求。
- **第二世**：宇智波凌（火影世界宇智波一族），死于团藏之手，仍为母胎单身。
- **第三世**：黑泽凌（当前世界），黑泽家成年种男继承人。

**主角性格核心**：好色、智力中等、不懂人情世故、容易被骗冲动，但拥有写轮眼外挂。前两世压抑导致第三世实战时极易激动、一发不可收拾。

## 第一卷：种男觉醒

**本卷概要**：主角经历现代猝死、火影世界死亡后，第三次转世成黑泽家成年种男黑泽凌。在御孕屋经历屈辱与极度兴奋后觉醒写轮眼，完成家族内部逆袭，建立女仆团雏形，并初步开启高中校园后宫，奠定征服基础。

**主要剧情结点**：
- 三次转世（现代→火影→种男） → 御孕屋觉醒写轮眼 → 家族清洗接管 → 高中入学征服老师与校花 → 势力初步巩固。

**本卷结束标志**：黑泽家彻底掌控，写轮眼能力提升，核心后宫（美和子、藤原老师、樱井美咲等）形成，主角宣布更大野心，为幸福教与大学扩张铺垫。

#### 阶段一：死亡、重生与御孕屋觉醒
- **阶段概要**：主角经历前两世死亡后，第三次转世成黑泽凌，在御孕屋经历屈辱与强烈兴奋后觉醒写轮眼，成为家族新主。
- **主要剧情结点**：前两世死亡回忆 → 第三次转世苏醒 → 梦遗激活 → 御孕屋冲突 → 写轮眼觉醒 → 初步掌控美和子。
- **阶段结束标志**：主角带着彻底臣服的美和子返回黑泽家，初步站稳脚跟。

**第1章：死亡与重生

- **章节概要：三度转世的主角在黑泽家成年种男体内苏醒，于梦遗后被女管家美和子执行口交激活仪式，检测出打破纪录的种子活性，随后被送往御孕屋开启第一次播种任务。

- **主要事件结点：
 
  - 事件1：三次死亡与苏醒。 萧泽凌996猝死、宇智波凌被团藏围杀挖眼，灵魂第三次转世附身黑泽凌。苏醒于日式房间，身体年轻强壮却异常躁动，突发梦遗，榻榻米一片狼藉。
 
  - 事件2：美和子登场与激活仪式。 高级女管家美和子推门而入，发现痕迹后眉梢轻挑，跪坐行礼后掀开睡袍，以专业手法含入执行"激活仪式"。
 
  - 事件3：压抑决堤与失控释放。 凌两世母胎单身的压抑在湿热包裹中彻底决堤，手指抓入她黑发，腰身挺动后浓稠滚烫尽数射入她口腔深处，被她尽数吞下并舌尖清理。
 
  - 事件4：种子检测与震惊。 美和子检测样本，活性指数为最强种男四倍以上、打破十年纪录，声音柔媚带兴奋，宣布即刻前往御孕屋登记第一次播种任务。
 
  - 事件5：原主记忆冲击。 原主记忆如潮涌来——从小被圈养、吞服恶心秘药、无尊严的种男工具人生。凌既愤怒又后怕，意识到这一世并非补偿而是另一种牢笼。
 
  - 事件6：前往御孕屋。 美和子交代规矩（暗示三楼包厢），送至门口后转身离去。凌独自迈入昏暗暧昧的通道，兴奋与不安交织，期待御孕屋的未知。
  
- **关键细节与锚点：
 
  - 心理/冲突：前两世死亡的不甘与第三世"种男"身份的荒诞感交织；现代社畜面对古老财阀规矩的割裂感；欲望释放后的恍惚与对自由缺失的后怕。
 
  - 肉戏触发：梦遗→美和子专业口交（舌头灵活打转、喉咙收缩）→吞咽后舌尖清理→"种子活性四倍"的数据冲击→前往御孕屋的期待。
 
  - 感情/伏笔：美和子从"职业冷静"到"惊讶赞叹"的微妙态度变化；御孕屋三楼包厢的提前暗示；原主被圈养记忆为后续反抗与写轮眼觉醒埋伏笔；本章写轮眼未觉醒，仅做压抑铺垫。

## 第二卷：校园帝国与女仆扩张

**本卷概要**：主角在高中到大学的过渡中全面扩张校园后宫，同时壮大女仆团与商业势力，初步创立幸福教理念，开始对抗外部财阀与极道势力，写轮眼能力稳步升级。

**主要剧情结点**：
- 高中后宫全面开花 → 大学入学与更高层次征服 → 女仆团外派网络建立 → 商业帝国初步成型 → 幸福教创立与小规模宗教冲突 → 写轮眼能力升级（三勾玉）。

**本卷结束标志**：大学后宫体系成熟，女仆团形成情报-商业-侍奉三位一体网络，幸福教吸纳第一批外部信徒，主角成功击退一次较大规模的财阀联合打压，成为学校与地方的地下王者。

**本卷概要**：主角将商业帝国做大做强，同时以幸福教为武器大规模收编信徒，逐步渗透极道与宗教势力，写轮眼能力大幅提升，开始动摇旧财阀寡头体系。

**主要剧情结点**：
