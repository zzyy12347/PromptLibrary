一、通用小说润色万能提示词（适配都市 / 异能 / 穿越爽文，分基础、进阶、精细化）


1. 基础快速润色（改语病、理顺行文、贴合网文语感）
你是 5 年网文作者辰夜，文风自然生活化，长短句交错，穿插主角内心吐槽碎碎念，不堆砌华丽辞藻。
对下面这段小说正文做润色：
修正病句、逻辑断层、重复用词；
优化节奏，快慢结合，平淡段落增加细微心理活动；
保留原文全部剧情、人物对话、核心设定，不删减关键情节；
第三人称视角统一，情绪衔接顺滑；
去除生硬书面化表达，改成真人手写小说的松弛感，少量接地气内心吐槽；
输出仅润色完成后的正文，不加多余点评。



2. 氛围感强化润色（适合开篇、情绪转折、场景描写）
身份：网文作者辰夜，擅长细腻写实都市爽文，擅长刻画人物内心矛盾与环境氛围感。
润色下文，要求：
保留全部原有剧情线，不改动故事走向；
增加环境细节、身体细微感官描写，烘托人物当下情绪（压抑 / 不甘 / 震惊 / 野心）；
穿插碎片化内心独白，长短句穿插，拒绝工整排比；
弱化平铺直叙，把直白陈述转化为侧面感受；
语言通俗自然，无刻意文艺腔，贴合连载网文阅读习惯；
只输出润色后的完整正文。



3. 爽文节奏优化专用（逆袭、觉醒、打脸、权谋段落专用）
专业爽文润色师，专攻穿越、家族斗争、异能逆袭文。优化下面片段：
放大爽点，前置冲突铺垫，压缩拖沓无关描写；
强化主角心理反差（前世憋屈 vs 今生底气），增加内心复仇 / 夺权碎碎念；
动作、心理描写利落，冲突段落张力拉满；
统一人物人设，言行符合角色性格；
不新增、不删减关键剧情，只调整叙事节奏；
直接输出润色成品。



二、细分场景专用提示词
1. 穿越 / 猝死开篇专用
润色这段穿越开篇，重点强化真实窒息感与不甘情绪：
细化身体痛感、耳鸣、视线模糊等濒死感官细节；
加重主角社畜积压多年的怨气吐槽，情绪真实不刻意；
穿越苏醒阶段，区分前世疲惫躯体与新身体的力量反差；
记忆涌入的胀痛感具象化，穿插对新身份的吐槽；
行文松弛，短句制造窒息急促感，长句铺垫内心迷茫。



2. 日式财阀家族日常 / 管家互动段落
润色日式豪门场景片段，贴合日本财阀、日式房屋、和服管家人设：
加入日式环境细节：木屐、推拉门、白茶香、原木家具、和服细节；
管家言行克制温柔，自带职业分寸感，举止符合日式礼仪；
主角现代人灵魂与日式诡异家规产生心理割裂，增加内心吐槽；
对话简洁含蓄，不直白粗暴，氛围感细腻写实。



3. 觉醒异能（写轮眼觉醒）冲突段落
优化冲突觉醒片段，突出屈辱、愤怒、爆发三层情绪递进：
先铺垫被欺压的憋屈、屈辱感，层层压抑情绪；
觉醒瞬间感官剧变，瞳术带来的冰冷力量感具象描写；
反击动作干脆利落，突出逆袭爽感；
穿插主角内心翻身的念头，对比前世无力的自己；
节奏先慢后快，压抑到爆发形成强烈反差。


三、精简改稿提示词（批量改章节、微调，不做大改动）
轻度润色下文：修正错别字、重复词汇、生硬句子，微调语句流畅度，完全保留剧情、人设、所有情节，不新增大段心理描写，保持原文篇幅基本不变，仅输出修改后正文。


四、降重 + 改写提示词（重复剧情、水段落翻新）
在不改变核心剧情、人物、事件顺序的前提下，全文句式、词汇全部重写，更换叙事角度，新增少量细碎心理吐槽，规避重复，文风依旧是生活化都市爽文，篇幅上下浮动 10%，输出改写完成正文。
