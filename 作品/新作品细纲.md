# 《轮回种王：写轮眼征服录》完整大纲

**作者备注**：模块化设计，方便连载写作与随时插入新点子。核心设定为成年穿越 + 从头高强度后宫 + 写轮眼血统征服流。

## 第一卷：种男觉醒

**本卷概要**：主角从现代死亡穿越成成年种男，在御孕屋觉醒写轮眼，完成家族内部逆袭，建立女仆团雏形，并初步开启高中校园后宫，奠定征服基础。

**主要剧情结点**：
- 死亡穿越 → 御孕屋觉醒写轮眼 → 家族清洗接管 → 高中入学征服老师与校花 → 势力初步巩固。

**本卷结束标志**：黑泽家彻底掌控，写轮眼能力提升，核心后宫（美和子、藤原老师、樱井美咲等）形成，主角宣布更大野心，为幸福教与大学扩张铺垫。

### 阶段划分

#### 阶段一：死亡、重生与御孕屋觉醒
- **阶段概要**：主角从现代死亡到穿越觉醒写轮眼，成为家族新主。
- **主要剧情结点**：死亡穿越 → 口交激活 → 御孕屋冲突 → 写轮眼觉醒 → 初步掌控美和子。
- **阶段结束标志**：主角带着美和子返回黑泽家，初步站稳脚跟。
**多线叙事**  
- **主线剧情**  
  - 事件1：现代死亡与灵魂穿越：林辰996猝死，灵魂穿越附身黑泽辰（成年种男）。
  - 事件2：苏醒与梦遗激活：主角苏醒后首次勃起并梦遗，被美和子（成熟御姐女管家）发现并执行“激活仪式”（口交吸精并品尝）。
  - 事件3：进入御孕屋：美和子赞叹主角精子活性极高，带他前往御孕屋，进入1楼大厅。
  - 事件4：御孕屋冲突：目睹淫乱大通铺场景，主角表达嫌恶，被看守大汉暴揍并强制按在满身精液的女人身上播种。
  - 事件5：写轮眼觉醒：极度屈辱与愤怒中血轮眼觉醒（单勾玉），反杀看守大汉。
  - 事件6：初步掌控美和子：用新觉醒的写轮眼控制/征服美和子，并在三楼包厢进行1V1深度调教。
  - 事件7：离开御孕屋：带着彻底臣服的美和子返回黑泽家本宅，阶段收尾。

- **支线**  
  - 事件1：主角内心吐槽现代社畜灵魂与“种男”身份的反差。
  - 事件2：简单了解黑泽家“种男”制度的残酷规则。
  - 事件3：御孕屋看守大汉对主角的轻视与侮辱（为后续打脸做铺垫）。
  
- **感情线**  
  - 事件1：苏醒后，美和子对“种子活性极高”的主角产生惊讶与职业性的兴趣。
  - 事件2：主角在御孕屋受辱时，美和子虽执行命令，但内心出现一丝复杂情绪（怜悯或好奇）。
  - 事件3：觉醒后主角强势反杀并用写轮眼控制她时，美和子从“服从任务”转变为“真正臣服”，产生敬畏与依恋的混合情感。
  - 事件4：三楼包厢调教过程中，美和子逐渐流露出主动侍奉的欲望与对主角的崇拜（初步建立主仆情感纽带）。
  
- **肉戏**  
  - 事件1：苏醒后的梦遗 + 美和子口交激活（详细感官描写，第一章高潮）。
  - 事件2：御孕屋强制播种（觉醒前简写，突出屈辱感）。
  - 事件3：觉醒后三楼包厢1V1深度调教美和子（高强度、征服向肉戏，主角第一次主动主导）。

- **突发奇想**  
  - 事件1：主角对御孕屋制度的吐槽或未来征服计划的初步想法。

- **阶段细纲使用建议：**  
  - 第1章：主线1+2 + 肉戏1 + 感情线1
  - 第2-3章：主线3+4 + 支线 + 肉戏2
  - 第4-5章：主线5+6 + 感情线3+4 + 肉戏3
  - 第6章：主线7 + 收尾

- **阶段结束标志**：主角正式掌握黑泽家部分实权。

- **阶段结束标志**：黑泽家内部彻底稳定。

#### 阶段五：卷末布局与升级
- **阶段概要**：总结第一卷，完成升级与铺垫。
- **主要剧情结点**：彻底掌控家族 → 写轮眼升级 → 野心宣言。

## 第二卷：校园帝国与女仆扩张

**本卷概要**：主角在高中到大学的过渡中全面扩张校园后宫，同时壮大女仆团与商业势力，初步创立幸福教理念，开始对抗外部财阀与极道势力，写轮眼能力稳步升级。

**主要剧情结点**：
- 高中后宫全面开花 → 大学入学与更高层次征服 → 女仆团外派网络建立 → 商业帝国初步成型 → 幸福教创立与小规模宗教冲突 → 写轮眼能力升级（三勾玉）。

**本卷结束标志**：大学后宫体系成熟，女仆团形成情报-商业-侍奉三位一体网络，幸福教吸纳第一批外部信徒，主角成功击退一次较大规模的财阀联合打压，成为学校与地方的地下王者。

## 第三卷：商业争霸与宗教扩张

**本卷概要**：主角将商业帝国做大做强，同时以幸福教为武器大规模收编信徒，逐步渗透极道与宗教势力，写轮眼能力大幅提升，开始动摇旧财阀寡头体系。

**主要剧情结点**：
- 大型商业并购与黑产彻底剥离 → 幸福教公开传播（福利+幻术双管齐下） → 收编极道组织 → 对抗传统宗教势力 → 大规模后宫盛典 → 与财阀寡头的正面冲突。

**本卷结束标志**：商业帝国初具规模，幸福教成为有影响力的新兴势力，极道与部分宗教势力被收编或重创，主角获得“日本地下之王”的初步地位，写轮眼接近完全体。

**本卷概要**：主角全面摧毁旧财阀寡头体系，整合日本地下与地上所有势力，建立以自己为核心的社会主义神国，实现从种男到幕后之王的终极逆袭，后宫与权力达到巅峰。
