# 《轮回种王：写轮眼征服录》世界设定

## 1. 核心世界观

这是一个**现代日本**背景的故事，表面是法治社会，暗面却被财阀寡头、极道势力、宗教组织共同操控。

主角**黑泽凌**经历三次转世：
- 第一世：现代社畜萧泽凌，996猝死。
- 第二世：宇智波凌，死于团藏之手。
- 第三世：黑泽家种男黑泽凌，觉醒写轮眼血脉。

他将从一个低贱的“种男”工具人，逐步逆袭成为掌控日本地下与地上的幕后之王，最终建立以自己为核心的“幸福教”与教会社会主义秩序。

**核心主题**：用写轮眼的精神控制能力 + 现代知识 + 组织制度，打破旧日本的极道、财阀、宗教三重枷锁。

## 2. 黑泽家据点：隐秘山村（核心设定）

黑泽家是四国偏远深山中的古老极道家族，表面以经营偏僻神社和温泉度假村为掩护，实际控制着一个**完全隐秘、自给自足的山村据点**（暂命名为“黑泽隐里”或“黑泽御影村”）。

### 山村主要设施：
- **种男养育屋**（主角苏醒地点）：位于村落核心区域的封闭日式庭院群。专门用于培育和圈养种男，从幼年起用祖传秘药进行身体强化。主角所在的成年种男区环境相对较好，但本质仍是牢笼。
- **孕妇区 / 繁育所**：与养育屋相邻，收容被播种后怀孕的女性（自愿或半强制）。提供医疗照料，但地位同样低下。
- **御孕屋**：村落中规模最大的地下娱乐与播种设施。伪装成高端温泉会所：
  - 1楼：大通铺集体播种区（公开淫乱）。
  - 2楼：半私密调教室。
  - 3楼：VIP包厢（高级种男或家族重要人物专用）。
  - 地下层：部分与人体实验基地连通。
- **地下人体实验基地**：位于村落最深处，专门进行秘药研发、血脉强化实验、以及部分黑产人体改造。黑泽家种男制度的药物强化技术即来源于此。
- **温泉区**：天然温泉资源，既是对外营业的掩护，也是内部放松与部分秘密仪式的场所（可作为后期女仆团培训或后宫场景地点）。

**山村特点**：
- 位置极度隐秘，四周被山林与结界（或现代监控+极道巡逻）包围，外人极难发现。
- 村内实行半封闭管理，种男与普通仆役行动受严格限制。
- 表面平静祥和（温泉、白茶、樱花），暗地充斥药物、欲望与权力斗争。

## 3. 写轮眼能力边界（硬规则）

- **单勾玉阶段**（第一卷）：可观察微表情、肌肉反应、瞳孔变化、行动轨迹；可植入简单幻觉和潜意识指令；近距离有明显心理压迫感。
- **禁止**：直接读心、远程扫描记忆、透视、凭空获取情报。
- **代价**：使用过多会导致眼痛、流鼻血、短暂昏迷（尤其在幼年期）。

写轮眼是主角的主要外挂，但不是万能。爽点来自“细节观察 + 幻术引导 + 组织布局”的组合，而不是直接开挂。

## 4. 黑泽家与种男制度

- “种男”是家族最底层的生育工具，从小被药物强化身体，地位极低，被称为编号而非名字。
- 凌作为第三次转世者，是家族紧急补充的成年种男，初期地位低下，需靠写轮眼逐步夺权。
- 家族通过秘药控制种男欲望与身体，同时用御孕屋完成“任务”以产生后代。

## 5. 女仆团体系

女仆团是黑泽家从极道家族向商业帝国、幸福教转型的核心干部组织，不是单纯的后宫。

- **功能**：人才培育、情报网络、行政管理、信仰传播、外放渗透。
- **管理层**：美和子是高级女管家，负责种男事务和女仆团培训，地位远高于普通种男。
- **发展路径**：从内宅侍奉 → 外放情报/商业 → 幸福教骨干。
- **原则**：忠诚第一，能力第二。通过教育、福利、幻术绑定形成组织凝聚力。

## 6. 旧黑产与转型

黑泽家旧黑产包括：
- 地下娱乐（御孕屋等）
- 人口控制与强制繁衍
- 非法医药与人体实验

主角的转型策略：
- **必须废除**：人口买卖、强制繁衍。
- **可改造**：地下会所转型为高端商务接待，医药黑产转为合法医疗，实验基地转向血脉/能力研究。
- **保留利用**：旧账册、罪证、关系网作为打击对手和洗白的武器。

转型核心：把“暴力 + 药物 + 买卖”的旧模式，改为“教育 + 福利 + 上升通道 + 精神控制”的新模式。

## 7. 主要势力

- **财阀寡头**：资本垄断者，主角最终要推翻的对象。
- **极道势力**：地盘争夺者，可收编或打击。
- **宗教势力**：信仰与洗钱渠道，主角以“幸福教”对抗。
- **幸福教**：主角后期建立的组织，用福利 + 幻术 + 教会社会主义对抗旧秩序。

## 8. 写作原则

- 初期主角地位低、处境危险（被困山村），需谨慎布局。
- 爽点来自“弱小身体 + 强大外挂”的反差，以及逐步逆袭的过程。
- 肉戏服务于征服与情感，不宜过度展开。
- 世界观以“制度变革 + 权力斗争”为主，色情为辅。
- 山村设定可作为第一卷主要舞台，后续随主角夺权逐步向外部扩张（高中、城市、大学等）。
