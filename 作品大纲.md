# 作品大纲（AI阅读优化版）

> 用途：本文件是全书主线推进索引。AI续写、扩写、改纲时，应先读取本文件，明确“故事要往哪里走”；再读取《世界设定.md》《角色塑造.md》《事件细纲.md》《女仆团体系.md》补细节。

## 0. 作品定位

- **暂定书名**：重生极道：开局写轮眼夺权
- **类型关键词**：现代日本、极道夺权、写轮眼幻术、无敌流、黑产洗白、组织建设、宗教战争、教会社会主义、幕后统治。
- **主角定位**：宇智波凌转生为黑泽凌，凭借残存查克拉与写轮眼，从黑泽家“优质种男”逆袭为黑泽家掌权者，最终建立幸福教并统治日本暗面。
- **核心爽点**：婴幼儿弱小外壳与宇智波灵魂的反差；写轮眼对现代人的降维压制；从极道黑产到宗教国家的组织升级。
- **叙事底色**：黑暗、掌控、制度改造、冷酷权谋。主角不走正义英雄路线，而是用更高阶的控制手段吞并旧秩序。

## 1. 一句话主线

主角黑泽凌以写轮眼夺取四国极道家族黑泽家的权力，收拢内宅与女仆团，剥离旧黑产、建立商业帝国，最终创立幸福教，以教会社会主义碾压财阀、极道和旧宗教，成为日本真正的幕后统治者。

## 2. 简介版本

前世，他是宇智波凌，知晓忍界剧情却败给人心，被团藏夺眼、被政客绞杀，最终在灭族之夜用伊邪那岐孤注一掷。

再次睁眼，他成了黑泽凌。

这里是现代日本四国深山的黑泽家族。神社鸟居下掩盖着人口贩卖、地下声色产业、御孕屋、神社地下实验室与制度化育种的罪恶。一出生，他就被判定为优质“种男”，本该成为黑泽家繁衍机器。

但这具婴儿身体里，住着一个曾俯瞰忍界的宇智波灵魂。

仅凭一丝查克拉与单勾玉写轮眼，他在半岁秋祭上向家主黑泽静江植入潜意识指令，从“牲口资产”变成“唯一继承人”。随后，他收拢黑泽千雪、美和子、绫乃、白川凛等内宅核心，建立女仆团制度，摸清黑泽家的所有黑产链条。

当他发现统一教与日本本土宗教的敛财逻辑时，黑泽凌意识到：极道的刀只能统治一时，信仰与福利才能统治一国。

于是，幸福教诞生。

这一次，他不谈正义，只谈掌控。

## 3. 主线引擎

全书主线由四条线共同推动：

| 主线 | 功能 | 推进方式 |
| --- | --- | --- |
| 夺权线 | 主角从资产变成统治者 | 幻术、账册、暗杀、人事替换、代行权接管 |
| 女仆团线 | 建立绝对忠诚的干部体系 | 储备计划、成人预备役、正式女仆、外放节点 |
| 黑产转型线 | 从极道黑产走向合法商业帝国 | 冻结高风险黑产、封存罪证、改造产业、洗白资金 |
| 幸福教线 | 从商业帝国升格为宗教国家 | 福利学校、信仰组织、媒体战、基层动员、反财阀战争 |

写作时，每个大事件最好至少推动其中两条线。例如：收服美和子同时推动“夺权线”和“女仆团线”；冻结跨县人口贩卖同时推动“夺权线”和“黑产转型线”。

## 5. 第一卷：潜龙在渊（黑泽村篇）

### 5.1 卷定位

第一卷是“极弱身体开局 + 内宅夺权 + 黑产摸底”的根基卷。重点不在出村扩张，而在黑泽村内部完成权力闭环。

### 5.2 起点状态

- 主角刚转生，身体是黑泽家男婴。
- 黑泽家把他判定为“优质种男”，未来将被圈养。
- 黑泽静江掌控家族、御孕屋、地下声色产业、神社地下实验室。
- 绫乃是静江隐藏暗刃。
- 美和子是静江旧部和主角乳母。
- 黑泽千雪是无害但重要的生母身份牌。

### 5.3 核心剧情

1. **半岁秋祭夺权**  
   主角以单勾玉写轮眼向黑泽静江植入潜意识，让自己从“种男资产”变成“唯一继承人”。

2. **绫乃前置线**  
   主角在婴幼儿期通过异常查克拉流动和行动轨迹发现绫乃，1-3岁进行浅层渗透，3岁毒宴前由绫乃密报毒计，7岁正式收网。

3. **内宅核心建立**  
   黑泽千雪成为名义吉祥物；美和子转为财务、情报、培训核心；绫乃成为暗刃；白川凛成长为内宅统筹继承人。

4. **女仆团制度化**  
   从身边侍女扩展为正式女仆、成人预备役、储备计划、外围亲属网络。

5. **黑产摸底**  
   主角通过账册、情报网、干部试探、写轮眼观察，摸清人口贩卖、御孕屋、神社地下实验室、地下声色产业、洗钱网络。

6. **代行权接管**  
   黑泽静江因长期幻术影响精神衰退，主角在12岁正式接管黑泽家代行权。

### 5.4 结尾状态

- 主角掌握黑泽家代行权。
- 女仆团制度雏形完成，向48人长期编制扩张。
- 神社地下实验室、御孕屋、地下声色产业等旧黑产已被摸清。
- 跨县人口贩卖被冻结。
- 洗白草案启动，但还没有真正大规模投资。
- 第二卷目标自然引出：剥离黑产、洗白上岸。

### 5.5 第一卷禁止跑偏

- 不要过早出村统一日本。
- 不要让财阀、全国政客、大型宗教势力过早正面登场。
- 不要把写轮眼写成读心术或全知扫描。
- 不要让未成年角色进入成人关系。
- 不要让第一卷就完成商业帝国，只能完成洗白准备。

## 6. 第二卷：剥茧抽丝（洗白上岸篇）

### 6.1 卷定位

第二卷是“旧黑产剥离 + 合法产业建立 + 外放网络成型”的转型卷。主角从黑泽村少主，变成四国地区新兴合法势力的幕后操盘者。

### 6.2 起点状态

- 主角已接管黑泽家代行权。
- 旧黑产仍在，但最危险的跨县人口贩卖已被冻结。
- 美和子、绫乃、白川凛分别掌握财务、武力、内务。
- 女仆团制度已成型，但外放网络刚起步。

### 6.3 核心剧情

1. **剥离黑产**  
   废除或封存人口买卖、御孕屋、神社地下实验室、旧地下声色产业中的高风险部分。

2. **罪证变资产**  
   将旧账册、实验室档案、贿赂名单、人口流向记录转化为控制干部和谈判的筹码。

3. **合法产业布局**  
   投资温泉疗养、地产、物流、医疗、教育、娱乐公司等表面合法产业。

4. **女仆团外放**  
   毕业成员进入大学、娱乐、商业、公关、法律、医疗、IT系统，成为黑泽家的外部节点。

5. **白川星奈明星线**  
   通过黑泽家控股娱乐公司，打造外务组标杆与储备计划广告样板。

6. **成年后宫扩张**  
   收服女警、商业对手女高管、政商圈关键女性等成年角色，扩展主角阵营。

### 6.4 结尾状态

- 黑泽家表面完成合法化。
- 合法商业帝国成型。
- 女仆团外放网络覆盖四国与部分全国节点。
- 幸福教的福利学校、基层组织、媒体包装开始出现。
- 统一教与本土宗教势力进入主角视野。

## 7. 第三卷：神国初现（幸福教篇）

### 7.1 卷定位

第三卷是“商业组织升格为宗教组织”的扩张卷。主角意识到，单纯商业无法真正统治底层，必须用信仰、福利、教育和组织力绑定群众。

### 7.2 核心剧情

1. **发现旧宗教敛财逻辑**  
   主角接触统一教、日本本土寺庙与腐败神社体系，看透宗教是比极道更高阶的统治工具。

2. **创立幸福教**  
   以商业帝国输血，以女仆团外放节点为干部，以写轮眼控制核心骨干，快速建立教会组织。

3. **教会社会主义路线**  
   不走传统邪教单纯敛财路线，而是提供教育、医疗、就业、债务重组、家庭援助等真实福利。

4. **基层组织扩张**  
   用福利学校、女性储备体系、社区互助、媒体宣传吸纳底层。

5. **宗教市场战争**  
   与统一教、本土寺庙、旧神社体系争夺信徒、媒体话语权、政界资源。

### 7.3 结尾状态

- 幸福教成为全国性新兴宗教组织。
- 主角掌握基层动员能力。
- 财阀寡头、本土寺庙、外来邪教开始联合警惕。
- 第四卷进入全国级别冲突。

## 8. 第四卷：龙战于野（统一日本篇）

### 8.1 卷定位

第四卷是“全国战争 + 旧秩序崩塌 + 幕后统治完成”的终局卷。

### 8.2 核心剧情

1. **财阀反扑**  
   幸福教的福利体系和基层组织威胁财阀利益，财阀开始联合媒体、政客、警方打压。

2. **旧宗教围剿**  
   本土寺庙、神社体系和外来邪教失去信徒与资金渠道，联手攻击幸福教。

3. **极道收编**  
   主角以黑泽家为核心，收编或摧毁其他极道组织，把暴力体系纳入暗影组与合法安保外壳。

4. **高级幻术升级**  
   万花筒写轮眼登场，主角通过更高阶幻术瓦解敌方核心人物的意志与组织信任。

5. **社会主义神国建立**  
   财阀寡头体系被击穿，幸福教成为日本基层、商业、宗教、政界背后的实际组织核心。

### 8.3 结尾状态

- 主角成为日本幕后真正统治者。
- 幸福教成为国家级组织。
- 旧极道、旧宗教、旧财阀被收编、瓦解或边缘化。
- 女仆团外放成员成为新秩序的行政、情报、媒体、医疗、教育节点。

## 10. 敏感与年龄线规则

- 18岁以下角色只写教育、保护、观察、精神绑定、权力归属。
- 成人关系、仪式、亲密服务、培训实操、外务应酬等内容统一引用《敏感留白.md》的 S-01 到 S-06。
- 不在大纲正文展开具体敏感过程。
- 写女仆团时优先强调组织功能：行政、情报、外务、安保、干部培养。

## 11. AI续写检查清单

续写任意章节前，先检查：

1. 当前情节属于哪一卷？
2. 这一章推动哪条主线：夺权、女仆团、黑产转型、幸福教？
3. 主角使用写轮眼是否符合能力边界？
4. 是否过早引入了下一卷才该正面登场的势力？
5. 是否把未成年角色写成了成人关系？
6. 是否把女仆团写成单纯陪衬，而忘了她们的组织功能？
7. 是否保留了“黑泽家 → 商业帝国 → 幸福教 → 幕后日本”的升级路径？
