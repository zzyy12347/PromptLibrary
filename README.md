# AI 小说提示词库桌面应用

这是一个本地运行的小说写作提示词管理工具，可以新增、编辑、删除、搜索提示词，并一键复制提示词内容。

## C++ 桌面版

双击 `run_cpp.bat` 启动 C++ 原生桌面版。

主界面左侧有“思维导图”按钮，可以打开导图管理窗口。

思维导图窗口支持：

- 上一页/下一页和下拉框切换不同导图
- 新建、删除、重命名导图
- 在左侧节点树里选择节点
- 修改节点标题和关联提示词标题
- 添加子节点、删除节点
- 在左侧节点树中拖拽节点调整结构：拖到目标上半区插到上面，中间放入目标下面，下半区插到后面
- 导入 JSON 导图文件，导出 Markdown 导图文件
- 单击画布节点选中，双击画布节点会在主界面定位到对应提示词

如果还没有编译，`run_cpp.bat` 会先调用 `build_cpp.bat` 编译，生成：

```text
x64\Release\PromptLibrary.exe
```

也可以双击 `PromptLibraryCpp.sln` 用 Visual Studio 打开工程。

## Python 旧版

之前的 Python 版本仍然保留。双击 `run.bat` 可以启动。

如果电脑没有把 Python 加入 PATH，可以在命令行里进入本文件夹后运行：

```bat
py app.py
```

## 数据说明

- 第一次启动时，应用会自动读取 `原创小说写作提示词库.md`，生成 `prompts.json`。
- 后续新增、修改、删除都会保存到 `prompts.json`。
- 搜索会匹配标题、分类、标签和提示词正文。
- 思维导图会保存到 `mindmaps.json`。

## EMMX 导图转换

`tools/import_emmx_mindmaps.py` 可以把指定的 `.emmx` 思维导图转换为 `mindmaps.json`。当前已导入：

- 世界设定
- 作品大纲
- 主角塑造
- 事件细纲

## 快捷键

- `Ctrl+F`：搜索
- `Ctrl+S`：保存当前提示词
- `Ctrl+N`：新建提示词
